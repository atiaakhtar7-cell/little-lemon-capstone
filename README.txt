Little Lemon Back-End Developer Capstone Project

API ENDPOINTS:

Menu API:
GET /api/menu/
GET /api/menu/<id>/
POST /api/menu/
PUT /api/menu/<id>/
DELETE /api/menu/<id>/

Booking API:
GET /api/bookings/
POST /api/bookings/
PUT /api/bookings/<id>/
DELETE /api/bookings/<id>/

User Authentication:
POST /api/registration/
POST /api/token/login/

Tested using Insomnia REST client.
Connected to MySQL database.
